# Multi Mouse Tool

Windows向けマルチマウス設定ユーティリティ。複数のマウスデバイスを自動検出し、デバイスごとに異なる設定を自動で切り替えます。

## インストール

<a href="https://apps.microsoft.com/detail/9MV0WCM2C3RF">
  <img src="https://get.microsoft.com/images/ja-jp%20dark.svg" alt="Microsoft Storeからダウンロード" width="200"/>
</a>

## 機能

- デバイスの自動検知・抜き差し時の自動更新
- 操作中デバイスのハイライト表示
- デバイス表示名のカスタマイズ
- デバイスの監視対象/監視外の切り替え
- デバイスごとのマウス速度設定
- Enhance Pointer Precision ON/OFF
- ボタン左右入れ替え（左利き対応）
- スクロールホイール速度設定
- 設定リセット機能
- タスクトレイ常駐
- 設定のJSON永続化

## 技術スタック

- Rust
- native-windows-gui (NWG)
- Win32 Raw Input API / SystemParametersInfo
